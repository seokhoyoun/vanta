# Vanta.Core

## 역할

모든 모듈이 공통으로 참조하는 가장 하위 기본 모듈입니다.

## 책임

- 공통 결과 타입
- 공통 예외 타입
- 시간, 식별자, 범용 유틸리티
- 모듈 전반에서 재사용되는 순수한 기본 코드

## 의존 규칙

- 다른 `Vanta.Comm.*` 모듈을 참조하지 않습니다.
- 가장 안정적인 코드만 위치시킵니다.

## 포팅 대상

레거시 `Library` 중에서도 통신 도메인에 직접 묶이지 않은 범용 유틸리티를 이관합니다.

## 현재 상태

- 모듈 앵커만 생성됨
- 실제 공통 타입은 아직 없음
