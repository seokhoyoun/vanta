# Vanta.Comm.Application

## 역할

통신 도메인의 유스케이스와 오케스트레이션을 담당하는 모듈입니다.

## 책임

- 런타임 시작, 정지 흐름
- 설정 로딩 유스케이스
- 장치, 블록, 태그, 프로세스, 시퀀스 동기화 흐름
- 애플리케이션 서비스
- 명령, 조회, 상태 집계

## 의존 규칙

- `Vanta.Core`, `Vanta.Comm.Contracts`, `Vanta.Comm.Abstractions`만 참조합니다.
- 인프라 세부 구현에 직접 의존하지 않습니다.

## 포팅 대상

레거시에서 `CommDaemon`이 직접 호출하던 실행 흐름과 `DLL_FRAME_ADAPTER`의 일부 조립 로직을 상위 애플리케이션 서비스로 재구성합니다.

## 현재 상태

- 모듈 앵커만 생성됨
- 초기 진입점은 "런타임 초기화"와 "구성 로드" 유스케이스가 될 예정
